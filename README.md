# [mkado@portfolio ~]$ ./my-cv

Personal portfolio & CV — terminal-themed, zero dependencies.

## Versions

| Path | Description |
|------|-------------|
| `/` | v1 — current portfolio |
| `/v3` | v3 — alternate version |

## Stack

- Vanilla HTML / CSS / JS
- JetBrains Mono (Google Fonts)
- Font Awesome 6 (CDN)
- IntersectionObserver API (scroll reveal + scroll spy)

## Features

- Responsive design (≥480px → ≥640px → desktop)
- Mobile hamburger nav with drawer
- Scroll-spy active nav highlighting
- Typewriter hero subtitle
- Project cards with hover reveal bar
- Terminal-style contact block

## Structure

```
my-cv/
├── index.html   ← v1 (root)
└── v3/
    └── index.html   ← v3
```
